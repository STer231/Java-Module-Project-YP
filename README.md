# Race Winner (Console App)

## О приложении
Небольшая консольная программа на Java, определяющая победителя гонки по скорости автомобилей.
Программа запрашивает у пользователя название и скорость трёх машин, рассчитывает дистанцию
и выводит имя победителя.

## Как запустить
Правой кнопкой по `Main` → **Run 'Main.main()'** (или Ctrl+Shift+F10).

## Пример работы
Введите название машины № 1

Введите скорость машины № 1

Введите название машины № 2

Введите скорость машины № 2

Введите название машины № 3  
BMW

Победитель гонки Ferrari

## Что отрабатывал в проекте
- Ввод/валидация пользовательского ввода (строки и целые числа).  
- Базовый ООП (классы Car, Race).  
- Логика сравнения/поиска максимума.  
- Работа с Scanner и бесконечными циклами для повторного запроса корректного ввода.
